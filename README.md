# OneBin. 🌿

OneBin is a premium, Swiss-inspired waste management education and tracking platform. It empowers users to manage their waste responsibly while earning rewards and competing with their community. Built with a focus on precision, functionality, and environmental impact.

## 🚀 Key Features

- **📊 Smart Dashboard:** Track your total earnings, waste deposits, and transaction history with clean, data-driven visualizations.
- **🔍 AI Waste Scanner:** Identify waste types instantly using Gemini AI to ensure correct disposal and maximize impact.
- **📍 Waste Bank Locator:** Find the nearest waste collection points and banks in your area using Google Maps integration.
- **🏆 Community Leaderboard:** Compete with others and see who's making the biggest impact on the environment.
- **💬 Community Feed:** Share your waste management journey, like posts, and engage with other eco-conscious users.
- **📚 Education Hub:** Learn about different types of waste and best practices for recycling and disposal through interactive quizzes.

## 🛠️ Tech Stack

- **Frontend:** [React](https://react.dev/) (Vite)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Backend/Database:** [Firebase](https://firebase.google.com/) (Authentication & Firestore)
- **AI Integration:** [Google Gemini AI](https://ai.google.dev/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Animations:** [Motion](https://motion.dev/)
- **Charts:** [Recharts](https://recharts.org/)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Firebase Project
- Google Gemini API Key

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd onebin
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Firebase and Gemini API configurations.
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   GEMINI_API_KEY=your_gemini_api_key
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy

OneBin follows a **Swiss Design** (International Typographic Style) aesthetic:
- **Bold Typography:** Utilizing **Inter** for maximum legibility and impact across all devices.
- **Grid-Based Layouts:** Precise alignment and structured information hierarchy for a professional feel.
- **Vibrant Accents:** Utilizing **Swiss Red** for primary branding and **Swiss Green** for positive actions and earnings.
- **Minimalism:** Focus on functionality and clarity, removing unnecessary clutter to highlight user data and impact.
- **Responsiveness:** A mobile-first approach ensuring a seamless experience from smartphones to ultra-wide monitors.
